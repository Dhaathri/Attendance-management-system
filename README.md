# 📌 Attendance Management System

A simple yet effective **Attendance Management System** built with **Python, Tkinter, and Pandas**.  
This project allows you to **mark attendance, store records in CSV, and view them in a table format** — all through a clean GUI.

## 🚀 Features
- ✅ **Mark Attendance** with student name, date, and status (Present/Absent).
- ✅ **CSV Integration**: Records are stored in `attendance.csv` for persistence.
- ✅ **View Records**: Opens a new window with a tabular view of all attendance data.
- ✅ **Error Handling**: Alerts if fields are left empty.
- ✅ **Reset Form**: Clears input fields after submission.
- ✅ **Lightweight GUI** using Tkinter.

## 🛠️ Tech Stack
- **Python**
- **Tkinter** (GUI framework)
- **Pandas** (CSV handling)
